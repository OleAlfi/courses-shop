# Academy - Courses Shop

A modern online course marketplace platform built with **Django 5.1+** and **Bootstrap 5.3**.

## Features
- **Course Catalog:** A clean, organized table displaying all active courses.
- **Dynamic Routing:** Individual pages for each course using Django's URL dispatcher.
- **Detailed Course View:** Deep-dive into course specifics (price, category, enrollment stats).
- **Responsive Design:** Fully optimized for mobile, tablet, and desktop viewing.

## Tech Stack
- **Backend:** Python 3.13, Django
- **Frontend:** HTML5, CSS3, Bootstrap 5.3
- **Database:** SQLite (Development standard)
- **IDE:** PyCharm Professional (macOS)

## How to run
1. Clone repository
2. Create new virtual env: `python -m venv venv`.
3. Install libraries: `pip install -r requirements.txt`.
4. Run server: `python manage.py runserver`.
